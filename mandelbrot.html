Mandelbrot

Instructions

Use SHIFT + MOUSE BUTTON to select a new center. Center coordinates are displayed in the first two boxes.

Use DOUBLE LEFT CLICK or MOUSE WHEEL UP to zoom in.

Use DOUBLE RIGHT CLICK or MOUSE WHEEL DOWN to zoom out. Zoom is displayed in the third box.

Restore default by using DEFAULT.

Activate coordinate behavior by using TOGGLE LINES.

To download or copy the image copy the link in the fourth box.

Change precision on ITERATIONS. WARNING: Higher iterations come with longer rendering times.
